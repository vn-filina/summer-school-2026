## 🛠️ Реализованные фичи MVP

В ходе разработки прототипа были успешно развернуты три продуктовые фичи:
1. **Динамическое ценообразование:** Автоматическое вычисление базовой стоимости на бэкенде в зависимости от типа мастер-класса (Ручная лепка — 1500 руб., Гончарный круг — 2000 руб.).
2. **Сидирование расписания:** Автогенерация сетки занятий на 180 дней вперед при первом запуске пустой базы данных.
3. **Идемпотентность API:** Защита от повторных случайных бронирований и овербукинга при сбоях сети с помощью механизма `Idempotency-Key` в заголовках запросов.

## 🚀 Быстрый запуск

### 1. Установка зависимостей
Убедитесь, что у вас установлен Python 3.10+, и выполните команду в терминале:
```bash
  pip install -r requirements.txt
```

### 2. Запуск Бэкенда (FastAPI)
Перейдите в папку бэкенда и запустите сервер:
```bash
  cd 01-analysis/5-prototype/backend
```
```bash
  uvicorn main:app --reload
```
Интерактивная документация API доступна по адресу: http://127.0.0.1:8000/docs

### 3. Запуск Фронтенда (Streamlit)
```bash
  streamlit run app.py
```
