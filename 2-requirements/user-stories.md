# User Stories — Гончарная мастерская «Глина»

> Контекст MVP: Система онлайн-записи для клиентов. Ограничение по ролям — только роль Клиент.

| ID | Роль | Функция | Бизнес-ценность |
| :-- | :-- | :-- | :-- |
| **US-001** | Клиент | Видеть расписание на 7 дней | Выбор удобного времени |
| **US-002** | Клиент | Видеть свободные места в слоте | Избегание овербукинга |
| **US-003** | Клиент | Записаться (Имя, Телефон, Прокат) | Бронирование места |
| **US-004** | Клиент | Отменить запись | Освобождение места для других |
